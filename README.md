# JSON-to-GeoJSON-Converter
A python code to convert JSON format to GeoJSON format

This code is run on VS code. Steps to be followed for easy execution. 
1. Create a file folder named "input" or "data", and "output" or "result" in the same folder as the code is saved.
2. Add JSON data to the "input" or "data" file.
3. Load the JSON data by setting the path. (mentioned at the end of the code)
4. Set the layer name. (for reference)
5. Set the path for the GeoJSON data to be saved.
6. Run the code.
